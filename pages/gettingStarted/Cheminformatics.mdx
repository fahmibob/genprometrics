## ğŸ““ Alur Notebook

Berikut struktur isi dan tujuan dari setiap notebook:

### 1. `01_koleksiData.ipynb`  
Mengimpor atau membangun kumpulan data senyawa dari file CSV, PubChem, atau input manual.

### 2. `02_lipinskiRo5.ipynb`  
Menghitung properti molekul dan mengevaluasi kelayakan senyawa berdasarkan **Lipinski's Rule of Five** untuk menentukan drug-likeness.

### 3. `03_filteringStructure.ipynb`  
Menerapkan penyaringan lanjutan berdasarkan sifat fisikokimia (seperti logP, PSA, berat molekul), serta deteksi struktur reaktif/PAINS.

### 4. `04_virtualScreening.ipynb`  
Mempersiapkan data molekul untuk proses **virtual screening**, termasuk konversi format dan visualisasi senyawa. Belum termasuk docking.
